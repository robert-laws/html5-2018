HTML - head tag

Title tag

Search engines use the title tag content to determine the contents of the webpage.

Meta tag

Typically, meta tags have a name and content pairing. The name parameter is used to describe the type of meta content and the content parameter is used for the information.

Examples of some meta tags include: description, keywords, robots, copyright, author, and charset (which is unique compared to the other types of meta tags - it declares the character set used, usually either utf-8 for western languages or utf-16 for languages with more than 256 characters).

The base tag

Used to define the default URL. There can only be one base tag in the entire html document.
