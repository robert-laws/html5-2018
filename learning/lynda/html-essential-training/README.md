## Structuring Content

Sectioning Elements - used to divide up an HTML page into distinct sections.

* `<h1>` ... `<h6>`

* `<article>`

* `<aside>`

* `<nav>`

* `<section>`

Semantic Elements - help add meaning to your document

* `<header>`

* `<main>`

* `<footer>`
